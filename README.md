# vue-template

Vue Project를 매번 설정하기 귀찮아 만드는 Vue Template

- @vue/cli 4.5.15
- Vue.js 3
- Typescript
- vue-router
- ESLint + Prettier
- Unit Test - Jest
- Snippets
- Git

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
